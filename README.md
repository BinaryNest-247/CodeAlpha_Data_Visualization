# 📊 Task 3 – Data Visualization  
CodeAlpha Data Analytics Internship

## 📌 Objective
The objective of this task is to transform raw data into meaningful visual representations to better understand patterns, trends, and insights.

## 📂 Dataset
The dataset used in this project was generated from the Web Scraping task (Task 1).  
It contains:
- Book Name
- Price

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib

## 🔎 Steps Performed

1. Loaded the dataset using Pandas.
2. Cleaned the "Price" column by removing the currency symbol.
3. Converted price values into numeric format.
4. Created visualizations to analyze pricing patterns.

## 📈 Visualizations Created

### 1️⃣ Histogram – Price Distribution
- Shows how book prices are distributed.
- Helps identify common price ranges.

### 2️⃣ Bar Chart – Top 10 Most Expensive Books
- Displays the highest priced books.
- Helps compare price differences visually.

## 💡 Key Insights
- Observed variation in book pricing.
- Identified the highest priced books.
- Understood overall price distribution pattern.

## 🎯 Outcome
This task improved my understanding of:
- Data storytelling
- Visual representation of numerical data
- Importance of visualization in decision-making
